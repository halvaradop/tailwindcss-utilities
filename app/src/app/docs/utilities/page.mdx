## Installation

Before using this package, ensure Tailwind CSS v4 is installed and configured. If needed, refer to the [official Tailwind installation guide](https://tailwindcss.com/docs/installation).

Once ready, install the package:

```bash
npm install -D @halvaradop/tailwindcss-utilities
# or
pnpm add -D @halvaradop/tailwindcss-utilities
```

</SectionLayout>

<Separator>Configuration</Separator>
<SectionLayout>

## Configuration

To use the utility classes from this package, add them to your Tailwind CSS setup as follows:

1. Make sure Tailwind CSS is imported in your main `.css` file using the `@import` directive.
2. Import the utility CSS file from this package after Tailwind.

Example:

```css
@import "tailwindcss";
@import "../node_modules/@halvaradop/tailwindcss-utilities/dist/plugin.css";
```

This setup ensures that the utility classes are available throughout your project.

</SectionLayout>

## Usage

This utility package offers a variety of class enhancements, such as:

- `text-fluid-{size}`
- `min-width-{screen}`
- `{html-selector}`
- `scrollbar`, `thumb`, `track`

### Example

```html
<div class="p:text-slate-200">
    <h1 class="text-fluid-2xl">Title</h1>
    <p>paragraph</p>
</div>
```

<Separator>Customization</Separator>
<SectionLayout>

### Adding Custom Values

Currently, the only utility type that supports extension and custom values is `text-fluid`. Future releases will allow more utilities to be customized as needed.

To define your own utility values, use the `@theme` directive in your `.css` file and add CSS variables with the appropriate prefixes. For example:

```css
@theme {
    /* Recommended: Use `clamp()` for responsive fluid sizes */
    --text-fluid-7xl: clamp(4.5rem, 9.2vw, 6.8rem);
    --text-fluid--line-height: 1.2;
    --text-fluid--letter-spacing: -0.038em;
}
```

Future versions will extend this customization to additional utilities.

</SectionLayout>
